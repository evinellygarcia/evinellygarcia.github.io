Card Suit Calculator

A C program developed to determine which player has the highest ranked hand of cards.

Description

This project analyzes card data and determines the winner based on predefined ranking rules. It supports reading from external files for scalability and handles edge cases such as invalid inputs. Dynamic memory allocation was used to ensure efficient system resource management.

Skills Learned

  • Determined which player has the highest ranked hand of cards.
  • Implemented logic to analyze card data and determine the winner based on predefined ranking rules.
  • Utilized file I/O operations for flexibility and scalability of input sizes.
  • Designed program to handle edge cases, ensuring robustness and reliability.
  • Implemented memory management with dynamic allocation to ensure efficient use of resources.
